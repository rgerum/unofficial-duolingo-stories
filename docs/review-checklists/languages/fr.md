# French review checklist

Extends `global.md` for courses teaching French (short code `fr`).

## Typography and orthography

- [ ] Space before `!`, `?`, `;`, `:` and inside « guillemets » (the editor
      lint flags the missing space mechanically; check that a *narrow no-break*
      space or no-break space is used, not a breaking space that can wrap).
- [ ] Elision is applied: l'homme, d'accord, j'ai, qu'il — never "le homme".
- [ ] Accents on capitals are kept (État, École, À demain).
- [ ] Days, months and languages are lowercase (lundi, janvier, français).
- [ ] Numbers: comma as decimal separator in running text.

## Grammar pitfalls to check

- [ ] Agreement of past participles (avoir + preceding direct object,
      être verbs, reflexives).
- [ ] Gender agreement of articles and adjectives.
- [ ] Negation is complete in written register (ne … pas) unless the story
      deliberately uses casual spoken style — then it should be consistent.
- [ ] Correct preposition with places (à Paris, en France, au Canada).

## Register and address

- [ ] tu/vous is consistent per character pair (strangers and formal
      situations: vous; friends/family/children: tu) and doesn't flip without
      a plot reason.
