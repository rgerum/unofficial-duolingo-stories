# Spanish review checklist

Extends `global.md` for courses teaching Spanish (short code `es`).

## Typography and orthography

- [ ] Questions and exclamations carry opening marks at the start of the
      question/exclamation itself, not necessarily the sentence:
      "Pero ¿qué haces?" (the editor lint flags missing `¿`/`¡` mechanically;
      check the *placement* here).
- [ ] Accents are present and correct, including on capital letters (Él, Ángel)
      and on interrogatives in indirect questions (qué, cómo, dónde).
- [ ] Days, months, languages and nationalities are lowercase (lunes, enero,
      español).
- [ ] Dialogue uses a consistent convention (em-dash — or quotes) across the
      whole story.

## Grammar pitfalls to check

- [ ] ser vs. estar chosen correctly for the context.
- [ ] Preterite vs. imperfect is consistent with the narration.
- [ ] Subjunctive used after triggers that require it (quiero que, ojalá,
      es importante que, …).
- [ ] Personal "a" before human direct objects (veo a María).
- [ ] Gender agreement between articles, nouns and adjectives.

## Register and address

- [ ] tú/usted/vosotros/ustedes usage is consistent per character pair and
      matches the course's variant (Latin American courses: ustedes, no
      vosotros).
